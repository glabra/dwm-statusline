# dwm-statusline
small status display utility for dwm window manager


## Dependencies
- libxcb
- alsa-lib


## Usage
`make && ./dwm-statusline`

Tested on Arch Linux on Xiaomi Mi Notebook Air 13.3.


## status line
`<CPU Temperture>C b<Brightness>% <Is Charging?><discharging/charging rate>/<Battery>% <datetime>`
